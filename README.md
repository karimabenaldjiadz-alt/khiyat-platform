# منصة خياط 🧵

منصة متكاملة للربط بين الزبائن والخياطين المحترفين.

## ✨ المميزات

- 🔐 تسجيل دخول وتسجيل مستخدمين (زبون/خياط)
- 🔍 البحث عن خياطين حسب التخصص والولاية
- 📦 إنشاء طلبات تصميم مع رفع صور ومقاسات
- 💰 تحديد السعر من قبل الخياط
- 💳 دفع إلكتروني (نموذج)
- 📊 متابعة حالة الطلب
- ⭐ تقييم الخياطين بعد الإكمال
- 📸 معرض أعمال للخياطين
- 🔔 نظام إشعارات فوري
- 💎 نقاط خبرة للخياطين

## 🛠️ التقنيات المستخدمة

- PHP 8.2
- MySQL / MariaDB
- HTML5, CSS3, JavaScript
- XAMPP (بيئة تطوير محلية)
mashro3i/
├── index.php # الصفحة الرئيسية
├── includes/ # الملفات المشتركة
├── pages/ # صفحات الموقع
│ ├── customer/ # صفحات الزبون
│ └── tailor/ # صفحات الخياط
├── assets/ # الملفات الثابتة (CSS, JS)
├── uploads/ # رفع الصور
└── sql/ # ملفات قاعدة البيانات

## 🚀 كيفية التشغيل

1. حمل XAMPP من [https://www.apachefriends.org/](https://www.apachefriends.org/)
2. انسخ مجلد `mashro3i` إلى `C:\xampp\htdocs\`
3. شغل Apache و MySQL من XAMPP Control Panel
4. افتح phpMyAdmin: `http://localhost/phpmyadmin`
5. أنشئ قاعدة بيانات `tailoring_db` واستورد ملف `sql/database.sql`
6. افتح المتصفح: `http://localhost/mashro3i/`
